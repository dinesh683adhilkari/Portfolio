# Dinesh Adhikari - Portfolio

Personal portfolio website built with Next.js 16, TypeScript, Tailwind CSS, shadcn/ui, Framer Motion.

## Features

- Responsive design
- Dark/Light theme
- Smooth animations
- Particle background
- Custom cursor
- Scroll progress
- EmailJS contact form (template_n2d2gu5)
- Project showcase

## Sections

- **Hero**: Animated typing effect, social links, resume download
- **About**: Bio, stats
- **Skills**: Interactive skill cards with progress bars
- **Projects**: Web & Data projects with thumbnails, tech stacks, links
