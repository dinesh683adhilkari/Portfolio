# Dinesh Adhikari - Portfolio

Modern personal portfolio website showcasing skills, projects, and experience.

## 🚀 Features

- **Next.js 16 + TypeScript** - App Router, Server Components
- **Tailwind CSS 4 + shadcn/ui** - Utility-first styling, accessible components
- **Framer Motion** - Smooth page transitions, scroll animations, hover effects
- **tsParticles** - Interactive particle background
- **Custom Cursor** - Enhanced UX with mouse-following cursor
- **Scroll Progress Bar** - Visual scroll indicator
- **Dark/Light Theme** - next-themes integration
- **EmailJS Contact Form** - Real form submissions (template_n2d2gu5)
- **Responsive Design** - Mobile-first, perfect on all devices
- **Loading Screen** - Smooth initial load experience
- **Glassmorphism UI** - Modern frosted glass effects

## 📱 Sections

- **Hero** - Animated typing titles, profile photo, CTA buttons, social links, resume download
- **About** - Personal bio, technical expertise, achievement stats
- **Skills** - Stacked interactive cards with progress bars (Frontend, Backend, Data/ML, Tools)
- **Projects** - Featured web & data projects with screenshots, tech stacks, GitHub/live demos
- **Experience** - Timeline of work/internships/leadership with achievements
- **Contact** - Form with EmailJS, contact info, location map placeholder
- **Footer** - Quick social links, scroll-to-top
